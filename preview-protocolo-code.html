¿Qué es exactamente una Tarifa CODE?

Una tarifa CODE es una tarifa de precio fijo y cerrado vinculada a un evento o colectivo concreto. Estas tarifas están diseñadas para facilitar la gestión de acuerdos específicos, asegurando que tanto el hotel como el cliente tengan las condiciones claras desde el momento de la reserva.

Promociones del Hotel

Creadas proactivamente por la dirección. Tienen como objetivo impulsar las ventas en momentos puntuales, premiar la fidelidad o lanzar campañas promocionales (ej. Black Friday, Oferta de Primavera).

Eventos Exclusivos (Solo Web)

Acuerdos específicos para organizadores y asistentes. Su comercialización se realiza exclusivamente a través de nuestra página web mediante un código promocional.

Estos casos son muy puntuales; la práctica habitual para eventos es ofrecer códigos con descuento sobre la tarifa BAR.

Características Generales

1
Exclusividad: cliente identificado o con código.
2
Tarifas netas: no generan comisión.
3
Condiciones fijas: precio, régimen y cancelación definidos.

Procedimiento en el PMS

1. Agencia: Seleccionar la agencia del evento (ej. “CINTURÓN DE LA MANCHA”).
2. Tarifa: Aparecerá automáticamente la tarifa CODE al seleccionar la agencia correcta.
3. Cancelación: Por defecto, cancelación 1 día antes, salvo acuerdo previo.
4. Pago: Según acuerdo comunicado (por defecto: pago en hotel).
5. Comisión: Tarifas netas, NO aplicar comisión.
6. Observaciones: Prohibido escribir la tarifa aquí; usar campos designados.
